# Golang Container
## _FullCycle_
A Golang container with size of 2MB.

## Building an image

This project requires [Docker](https://www.docker.com/docker-community).

To build the project Image, run the following command:

```sh
docker-compose build app
```

Running the container:

```sh
docker-compose up
```
